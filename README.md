# Textlite

A text-based adventure RPG game written in Python. _Text-based_ may be a little misleading, the game takes place entirely within the terminal it runs in. Textlite is designed to be as small and self-contained as is reasonably possible for a Python project, this also means there are few external dependencies.

This project is open to external contributions. If you come across issues in game, or in the code, let me know! Or you can fix them yourself and upstream them into this repo and become a contributor to the project!
